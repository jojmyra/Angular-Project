นโยบายความเป็นส่วนตัวและนโยบาย COOKIE ของ Space Run

หากคุณอาศัยอยู่ในประเทศดังต่อไปนี้ คุณต้องปฏิบัติตามข้อตกลงเฉพาะสำหรับประเทศต่างๆ บางประการ
          ทวีปอเมริกา: ประเทศอาร์เจนตินา
          ทวีปเอเชีย: ประเทศเกาหลี
          ทวีปยูเรเซีย: ประเทศรัสเซีย, ประเทศตุรกี
ทวีปตะวันออกกลาง: ประเทศบาห์เรน,ประเทศคูเวต, ประเทศเลบานอน, ประเทศโอมาน, ประเทศการ์ตาร์, ประเทศซาอุดิอาระเบีย, ประเทศสหรัฐอาหรับเอมิเรตส์
นโยบายความเป็นส่วนตัวนี้ (“นโยบาย”) อธิบายถึงข้อมูลส่วนบุคคลที่ถูกเก็บ หรือสร้างขึ้น (ประมวลผล) เมื่อคุณใช้เว็บไซต์ของ Space Run (“เว็บไซต์”) ตัวอย่างเช่น Space Run.com และ Space Runplus.Space Run.com และแอพพลิเคชั่นบนมือถือของ Space Run (“แอพพลิเคชั่น”) นโยบายนี้อธิบายถึงประเภทของข้อมูลส่วนบุคคลที่ถูกเก็บเมื่อคุณใช้เว็บไซต์หรือแอพพลิเคชั่นของเรา และข้อมูลส่วนบุคคลนั้นถูกใช้, แชร์ และป้องกันอย่างไร รวมถึงอธิบายว่าคุณมีทางเลือกใดบ้างเกี่ยวกับข้อมูลส่วนบุคคลของคุณและวิธีการติดต่อเรา หากคุณอาศัยอยู่ในประเทศบาห์เรน, ประเทศคูเวต, ประเทศเลบานอน, ประเทศโอมาน, ประเทศการ์ตาร์, ประเทศรัสเซีย, ประเทศซาอุดิอาระเบีย, ประเทศตุรกี หรือ ประเทศสหรัฐอาหรับเอมิเรตส์ โปรดพิจารณาข้อตกลงเฉพาะของประเทศต่างๆ
ใครเป็นผู้รับผิดชอบการประมวลผลข้อมูลส่วนบุคคลของคุณ?
นิติบุคคลของ Space Run ที่เป็นผู้รับผิดชอบในการประมวลผลข้อมูลส่วนบุคคลของคุณนั้น ขึ้นอยู่กับวิธีการใช้เว็บไซต์และแอพพลิเคชั่นของเรา และพื้นที่ที่คุณอาศัยอยู่
บริษัทในเครือของ Space Run ที่เกี่ยวข้องและ ไนกี้ อิงค์ จะถูกเรียกว่า “Space Run” หรือ “เรา” โปรดพิจารณา รายชื่อนิติบุคคลท้องถิ่น ของเราสำหรับชื่อนิติบุคคลของ Space Run ผู้รับผิดชอบและข้อมูลในการติดต่อเรา
ข้อมูลส่วนบุคคลใดที่เราเก็บและเก็บเมื่อใด
เราขอข้อมูลส่วนบุคคลบางส่วนของคุณเพื่อจัดหาสินค้าหรือบริการที่คุณขอให้แก่คุณ ตัวอย่างเช่น เมื่อคุณทำการซื้อ, ติดต่อศูนย์บริการผู้บริโภคของเรา, ขอรับการติดต่อสื่อสาร, สร้างบัญชี, เข้าร่วมกิจกรรมหรือการแข่งขันของเรา หรือใช้เว็บไซต์หรือแอพพลิเคชั่นของเรา ทั้งนี้ ข้อมูลส่วนบุคคลนี้รวมถึง:
    • รายละเอียดสำหรับการติดต่อ รวมถึง ชื่อ, อีเมล, เบอร์โทรศัพท์ และที่อยู่ในการจัดส่งและเรียกชำระเงิน
    • รายละเอียดการเข้าสู่ระบบและบัญชี รวมถึง ชื่อที่ปรากฎบนหน้าจอ, รหัสผ่าน และ ID ผู้ใช้งาน
    • ข้อมูลส่วนตัว รวมถึง เพศ, ที่อยู่, วันเกิด, ประวัติการซื้อ
    • รายละเอียดการชำระเงิน หรือบัตรเครดิต
    • รูปภาพ, รูปถ่าย และวิดีโอ
    • ข้อมูลเกี่ยวกับลักษณะทางกายของคุณ รวมถึง น้ำหนัก, ส่วนสูง, สัดส่วนของร่างกาย (เช่น ระยะการก้าว, ขนาดรองเท้า/เท้า, ขนาดเสื้อชั้นใน)
    • ข้อมูลการออกกำลังกายซึ่งคุณส่งมอบให้หรือซึ่งถูกสร้างขึ้นผ่านเว็บไซต์หรือแอพพลิเคชั่นของเรา (เวลา, ระยะเวลา, ระยะทาง, สถานที่, การนับแคลอรี่, FuelPoints, ความเร็วในการวิ่ง)
    • ความพึงพอใจส่วนตัว รวมถึงรายการสินค้าที่ต้องการ, โฆษณา และ Cookie ที่เลือกไว้
เราใช้ข้อมูลส่วนบุคคลเพิ่มเติมเพื่อเปิดการใช้งานคุณลักษณะเฉพาะบางอย่างบนเว็บไซต์และแอพพลิเคชั่น หรือบริการของเรา ตัวอย่างเช่น เราขอเข้าถึงข้อมูลตำแหน่งทางภูมิศาสตร์ผ่านทางโทรศัพท์ของคุณเพื่อบันทึกเส้นทางการวิ่งของคุณ, ข้อมูลรายชื่อผู้ติดต่อเพื่อให้คุณสามารถมีปฏิสัมพันธ์กับเพื่อนได้, ข้อมูลปฏิทินของคุณเพื่อจัดตารางเวลาแผนการออกกำลังกายของคุณ หรือข้อมูลเครือข่ายสังคมออนไลน์ของคุณเพื่อโพสต์เนื้อหาจากแอพพลิเคชั่นลงบนเครือข่ายสังคมออนไลน์ ทั้งนี้ ข้อมูลส่วนบุคคลนี้รวมถึง:
    • ข้อมูลการเคลื่อนไหวจากเครื่องวัดความเคลื่อนไหวของคุณ
    • รูปภาพ, รายชื่อผู้ติดต่อ และข้อมูลปฏิทิน
    • ข้อมูลจากระบบเซ็นเซอร์ รวมถึงอัตราการเต้นของหัวใจและตำแหน่งที่ตั้ง (GPS)
    • ข้อมูลเครือข่ายสังคมออนไลน์ รวมถึงหลักฐานอ้างอิงและข้อมูลใดๆ จากโพสต์สาธารณะของคุณเกี่ยวกับ Space Run หรือการสื่อสารระหว่างคุณกับเรา
เมื่อคุณใช้เว็บไซต์หรือแอพพลิเคชั่นของเรา ข้อมูลจะถูกเก็บและแชร์ให้ Space Run โดยอัตโนมัติด้วยเทคโนโลยีแพลตฟอร์มที่ให้บริการประสบการณ์ ตัวอย่างเช่น เว็บบราวเซอร์หรืออุปกรณ์มือถือของคุณอาจแชร์ข้อมูลบางอย่างให้ Space Run เมื่อมีการใช้งานเว็บไซต์หรือแอพพลิเคชั่นของ Space Run บนอุปกรณ์นั้นๆ รายละเอียดเพิ่มเติมในเรื่องนี้ได้มีการรวบรวมไว้ในหมวด Cookies and Pixel Tags ด้านล้าง ทั้งนี้ ข้อมูลดังกล่าวรวมถึง:
    • ID ของอุปกรณ์, Call State, การเข้าถึงระบบอินเทอร์เน็ต, รายละเอียดแหล่งเก็บข้อมูล และข้อมูลเกี่ยวกับแบตเตอรี่
    • Cookies, IP Addresses, Referrer Header, ข้อมูลบ่งบอกเว็บบราวเซอร์และรุ่นของคุณ และ Web Beacons และ Tags 
เครื่องมือที่ใช้ในการจัดการกับข้อมูลที่เรารวบรวม
ในการใช้แอพพลิเคชั่นของเรา เราจะทำการแจ้งคุณเกี่ยวกับการรวบรวมและการใช้ของเราผ่านการอนุญาตพื้นฐานบนร้านค้าแอพพลิเคชั่น, ระบบการปฏิบัติงานของโทรศัพท์มือถือ และประสบการณ์ในแอพพลิเคชั่น นอกจากนี้ เรายังมีแจ้งเตือนเพื่อขอความยินยอมจากคุณเมื่อจำเป็น
ในหลายกรณี เว็บบราวเซอร์หรือแพล็ตฟอร์มบนอุปกรณ์มือถือของคุณจะจัดหาเครื่องมือเพิ่มเติมเพื่อให้คุณสามารถควบคุมได้ว่าจะให้อุปกรณ์ของคุณรวบรวมหรือแชร์ข้อมูลบางประเภทตอนไหน ตัวอย่างเช่น อุปกรณ์มือถือหรือเว็บบราวเซอร์ของคุณอาจเสนอเครื่องมือเพื่อให้คุณสามารถจัดการการใช้งาน Cookie หรือการแชร์ตำแหน่งที่ตั้ง
เราสนับสนุนให้คุณทำความคุ้นเคยและใช้เครื่องมือเหล่านี้บนอุปกรณ์ของคุณ
ทำไมเราถึงใช้ (ประมวลผล) ข้อมูลส่วนบุคคลของคุณและอย่างไร
เราใช้ข้อมูลส่วนบุคคลซึ่งรวบรวมมาจากคุณเพื่อการดังนี้:
เพื่อจัดหาคุณลักษณะเฉพาะของเว็บไซต์ แอพพลิเคชั่นและบริการที่คุณขอ
เมื่อคุณใช้เว็บไซต์และแอพพลิเคชั่นของเรา เราจะอาศัยข้อมูลของคุณในการจัดการสินค้าและบริการที่คุณเลือกไว้ ตัวอย่างเช่น หากคุณทำการซื้อสินค้าบน Space Run.com หรือเข้าร่วมกิจกรรมหรือโปรโมชั่น เราจะใช้ข้อมูลการติดต่อที่คุณให้ไว้ในการติดต่อกับคุณเพื่อแจ้งเกี่ยวกับการซื้อ กิจกรรม หรือ โปรโมชั่น หากคุณติดต่อไปยังศูนย์บริการลูกค้าของเรา
เราจะใช้ข้อมูลของคุณ อาทิ รายละเอียดการจัดส่งหรือการชำระเงิน หรือสินค้าที่คุณซื้อเพื่อช่วยเหลือคุณแก้ไขปัญหาหรือคลายข้อสงสัยให้แก่คุณ
หากคุณใช้แอพพลิเคชั่นของเราเพื่อการติดตามการออกกำลังกายหรือลักษณะทางกายของคุณ เราจะรวบรวมข้อมูลนี้และจัดเก็บไว้เพื่อให้คุณสามารถเข้ามาดูได้ในแอพพลิเคชั่น ข้อมูลการออกกำลังกายของคุณอาจรวมถึงข้อมูลที่คุณป้อนเองหรือที่ถูกจัดเก็บผ่านอุปกรณ์ของคุณระหว่างกิจกรรมนั้น เช่น ข้อมูลตำแหน่งที่ตั้งหรือข้อมูลการเคลื่อนไหว เราอาจใช้ข้อมูลกิจกรรมเหล่านี้เพื่อคำนวณข้อมูลเกี่ยวกับกิจกรรมของคุณในอนาคต เช่น ระยะทางการวิ่ง หรือการเผาผลาญแคลอรี่ เพื่อให้คุณได้เข้าถึงข้อมูลที่ผ่านการคำนวณนี้ซึ่งเป็นส่วนหนึ่งในการทำงานของแอพพลิเคชั่น
ในหลายกรณี ในการใช้คุณลักษณะเฉพาะบางอย่างภายในเว็บไซต์ แอพพลิเคชั่น หรือบริการของเรา คุณอาจต้องให้ข้อมูลเพิ่มเติมหรือความยินยอมเพิ่มเติมแก่ Space Run ในการใช้ข้อมูลบางอย่างด้วยวิธีบางประการ ตัวอย่างเช่น ในการจะใช้คุณลักษณะเฉพาะในการวัดอัตราการเต้นของหัวใจ คุณอาจต้องเชื่อมต่ออุปกรณ์วัดอัตราการเต้นของหัวใจกับประสบการณ์; ในการแชร์ข้อมูลลงบนสังคมออนไลน์คุณอาจต้องเปิดเผยบัญชีสังคมออนไลน์ของคุณ
เพื่อติดต่อสื่อสารกับคุณเกี่ยวกับข้อมูลสินค้า บริการ กิจกรรม และโปรโมชั่นของเรา
ในการแก้ไขการติดตาม (อีเมลหรือเอสเอ็มเอส) ของคุณ หรือในการเลือกไม่รับการสื่อสารจาก Space Run กรุณาแก้ไขความพึงพอใจของคุณที่หัวข้อ “ดูหรือเปลี่ยนโปรไฟล์ของฉัน” บนเว็บไซต์ของเรา คุณยังสามารถติดต่อเราเพื่อใช้สิทธิเกี่ยวข้อมูลส่วนบุคคลของคุณผ่านรายละเอียดติดต่อที่หัวข้อ “คำถามและผลตอบรับ” หรือส่งอีเมลไปที่ privacy@Space Run.com
หากคุณเป็นลูกค้าปัจจุบันของ Space Run (ตัวอย่างเช่น หากคุณได้เคยทำการซื้อสินค้ากับเรา) เราอาจใช้อีเมลที่คุณให้มาในการส่งโฆษณาเกี่ยวกับสินค้า Space Run หรือบริการที่คล้ายกันให้คุณ เว้นแต่คุณเลือกที่จะไม่รับโฆษณาดังกล่าว ในกรณีอื่น ๆ เราขอความยินยอมจากคุณในการส่งโฆษณาเกี่ยวกับรายละเอียดของสินค้า บริการ กิจกรรมและโปรโมชั่นของ Space Run เราอาจใช้ข้อมูลที่คุณให้แก่เรา และข้อมูลจากสินค้าหรือบริการของ Space Run เช่น การใช้เว็บไซต์และแอพพลิเคชั่นของ Space Run ของคุณ, การเข้าชมหรือการซื้อสินค้าในร้าน Space Run ของคุณ, การเข้าร่วมกิจกรรมและการแข่งขันต่างๆ ของคุณ (ซึ่งอาจจะผ่านทางบริษัทในเครือ Space Run) เพื่อตั้งค่าการติดต่อสื่อสารเกี่ยวกับสินค้าและบริการที่น่าสนใจเฉพาะสำหรับคุณ หากมีกฎหมายบังคับไว้ เราจะขอความยินยอมจากคุณในการส่งโฆษณาให้แก่คุณ
เพื่อปฏิบัติการ พัฒนา และรักษาธุรกิจ สินค้า และบริการของเรา
เราอาจใช้ข้อมูลที่คุณให้ในการดำเนินธุรกิจของเรา ตัวอย่างเช่น เมื่อคุณทำการซื้อ เราอาจะใช้ข้อมูลนั้นในการทำบัญชี ตรวจสอบบัญชี และการดำเนินงานภายในอื่น ๆ อีกตัวอย่างหนึ่งเช่น เราอาจใช้ข้อมูลเกี่ยวกับวิธีการใช้สินค้าหรือบริการของคุณเพื่อพัฒนาประสบการณ์การใช้งานของคุณ และเพื่อให้เราวิเคราะห์ปัญหาทางเทคนิคหรือบริการและบริหารเว็บไซต์และแอพพลิเคชั่นของเรา
เพื่อคุ้มครองสิทธิ ทรัพย์สิน หรือ ความปลอดภัยของเราและผู้อื่น
เราอาจใช้ข้อมูลการใช้งานเว็บไซต์และแอพพลิเคชั่นของคุณเพื่อป้องกันหรือตรวจจับการทุจริต, ละเมิด, การใช้งานที่ผิดกฎหมายและการฝ่าฝืนข้อตกลงการใช้งานของเรา และเพื่อปฏิบัติตามคำสั่งศาล, รัฐบาล หรือตามกฎหมายที่ใช้บังคับอื่นๆ
เพื่อการค้นคว้าทั่วไป และเพื่อวัตถุประสงค์ในการวิเคราะห์
เราใช้ข้อมูลการใช้งานเว็บไซต์ แอพพลิเคชั่นและบริการของผู้เยี่ยมชมเพื่อให้เข้าจพฤติกรรมและความพึงพอใจของลูกค้าของเรา ตัวอย่างเช่น เราอาจใช้ข้อมูลเกี่ยวกับการค้นหาสินค้าของผู้เยี่ยมชม Space Run.com เพื่อให้เราทราบว่าจะใช้วิธีใดในการจัดวางสินค้าหรือเสนอสินค้าที่หน้าร้านของเราให้ดีที่สุด 
การใช้ (ประมวลผล) Workout Info  
ดังที่ได้กล่าวไว้ข้างต้น Space Run รวบรวมข้อมูลเกี่ยวกับกิจกรรมการออกกำลังกายหรือลักษณะทางกายภาพของคุณ (ซึ่งรวมกันเรียกว่า Workout Info) เพื่อส่งมอบแอพพลิเคชั่นและบริการอื่นของเรา ด้วยลักษณะของความเป็นส่วนตัวของข้อมูลเหล่านี้ เราจึงมุ่งมั่นที่จะชี้แจงข้อมูลเกี่ยวกับวิธีการใช้งาน Workout Info ให้แก่คุณอย่างชัดเจน เนื่องจากข้อมูลเหล่านี้อาจเป็นข้อมูลที่มีความอ่อนไหวในบางเขตอำนาจศาล เราจึงใช้มาตรการที่เหมาะสมในการคุ้มครองและใช้ข้อมูลเหล่านี้ และเราจะขอความยินยอมของคุณเพื่อการใช้ Workout Info ในกรณีที่มีกฎหมายบังคับหรือเพื่อให้เป็นไปตามนโยบายภายในของ Space Run เราจะขอความยินยอมของคุณเพื่อการใช้ Workout Info คลิกที่นี่ FAQ on Workout Info เพื่อข้อมูลเพิ่มเติม
เพื่อวัตถุประสงค์อื่นๆ
เราอาจใช้ข้อมูลส่วนบุคคลของคุณในทางอื่น ๆ และจะแจ้งให้คุณทราบเป็นการเฉพาะหากจะมีการรวบรวมข้อมูล รวมถึงจะขอความยินยอมจากคุณเมื่อจำเป็น
การแชร์ข้อมูลส่วนบุคคลของคุณ
การแชร์ของ Space Run
Space Run แชร์ข้อมูลส่วนบุคคลของคุณกับ:
    • บริษัทในเครือของ Space Run เพื่อวัตถุประสงค์และภายใต้เงื่อนไขดังที่ได้กล่าวมาข้างต้น
    • บุคคลที่สามซึ่งเป็นผู้ให้บริการและทำการประมวลผลข้อมูลส่วนบุคคลในนามของ Space Run ตัวอย่างเช่น เพื่อดำเนินการเกี่ยวกับบัตรเครดิตและการชำระเงิน, การจัดส่งสินค้า, การจัดการและบริการข้อมูล, ส่งอีเมล,
      การค้นคว้าและวิเคราะห์, การจัดการแบรนด์และโปรโมชั่นสินค้า รวมไปถึงการบริหารจัดการการบริการและคุณลักษณะเฉพาะบางอย่าง
    • บุคคลที่สามอื่นๆ เท่าที่จำเป็นเพื่อ: (i) ปฏิบัติตามคำสั่งของรัฐบาล, คำสั่งศาล หรือกฎหมายที่ใช้บังคับ; (ii) ป้องกันการใช้งานอย่างผิดกฎหมายบนเว็บไซต์และแอพพลิเคชั่นของเรา หรือการฝ่าฝืนข้อตกลงการใช้งานเว็บไซต์และแอพพลิเคชั่นและนโยบายของเรา; (iii) ปกป้องเราจากข้อเรียกร้องของบุคคลที่สาม; และ (iv) ช่วยเหลือในการป้องกันการทุจริตหรือการสืบสวน (เช่น การปลอมแปลง)
นอกจากนี้ เราอาจโอนข้อมูลส่วนบุคคลที่เรามีเกี่ยวกับคุณในกรณีที่เราขายหรือโอนกิจการหรือทรัพย์สินทั้งหมดหรือบางส่วน (รวมไปถึงกรณีของการปฏิรูปโครงสร้าง, แยกบริษัท, เลิกบริษัท หรือชำระบัญชี) หากคุณอาศัยอยู่ในประเทศเกาหลี โปรดพิจารณาข้อตกลงเฉพาะของประเทศต่างๆ
การแชร์ของคุณ
เมื่อคุณใช้คุณลักษณะเฉพาะเกี่ยวกับสังคมบางอย่างบนเว็บไซต์หรือแอพพลิเคชั่นของเรา คุณสามารถสร้างโปรไฟล์สาธารณะซึ่งอาจระบุข้อมูลจำพวก ชื่อที่ปรากฎบนหน้าจอ, รูปโปรไฟล์ และที่อยู่ คุณยังสามารถแชร์เนื้อหาดังกล่าวกับเพื่อนหรือต่อสาธารณะ รวมไปถึงข้อมูลเกี่ยวกับกิจกรรม Space Run ของคุณด้วย
เราสนับสนุนให้คุณใช้เครื่องมือที่เรามีให้เพื่อจัดการการแชร์ดังกล่าวเพื่อควบคุมข้อมูลที่คุณจะเปิดเผยบนคุณลักษณะเฉพาะเกี่ยวกับสังคมของ Space Run
การคุ้มครองและจัดการข้อมูลส่วนบุคคลของคุณ
    • การเข้ารหัสและความปลอดภัย: เราใช้มาตรการความปลอดภัยทางเทคนิคและการจัดระเบียบหลากหลายรูปแบบ รวมถึงการเข้ารหัสและเครื่องมือรับรองความแท้จริง เพื่อรักษาความปลอดภัยของข้อมูลส่วนบุคคลของคุณ ข้อมูลส่วนบุคคลของคุณจะถูกเก็บไว้บนเครือข่ายที่ปลอดภัยและเข้าถึงได้เฉพาะจากกลุ่มคนที่มีสิทธิพิเศษในการเข้าถึงระบบเหล่านี้เท่านั้น
การโอนข้อมูลระหว่างประเทศ: ข้อมูลส่วนบุคคลของคุณที่เรารวบรวมหรือสร้างขึ้น (ประมวลผล) บนพื้นที่เว็บไซต์และแอพพลิเคชั่นของเราจะถูกเก็บรักษาไว้ในประเทศสหรัฐอเมริกา ผู้รับโอนข้อมูลบางส่วนซึ่ง Space Run ได้แชร์ข้อมูลส่วนบุคคลของคุณด้วยอาจอาศัยอยู่ในประเทศนอกเหนือจากประเทศที่ข้อมูลส่วนบุคคลของคุณถูกรวบรวม กฎหมายในประเทศเหล่านั้นอาจไม่สามารถคุ้มครองข้อมูลได้ในระดับเดียวกับกฎหมายของประเทศต้นทางข้อมูล อย่างไรก็ดี เมื่อเราโอนข้อมูลส่วนบุคคลของคุณไปยังผู้รับในประเทศอื่นรวมถึงประเทศสหรัฐอเมริกา เราจะคุ้มครองข้อมูลส่วนบุคคลของคุณตามนโยบายนี้และกฎหมายที่ใช้บังคับ
หากคุณอาศัยอยู่ในเขตเศรษฐกิจยุโรป (“EEA”) หรือประเทศสวิตเซอร์แลนด์ เราใช้มาตรการในการปฏิบัติตามข้อกำหนดของกฎหมายที่ใช้บังคับและจัดให้มีความคุ้มครองที่เหมาะสมในการโอนข้อมูลส่วนบุคคลของคุณไปยังผู้รับโอนภายนอก EEA หรือ ประเทศสวิตเซอร์แลนด์ หากคุณอาศัยอยู่ในประเทศรัสเซีย โปรดพิจารณาข้อตกลงเฉพาะของประเทศต่างๆ
    • การกักข้อมูลของคุณ: เราจะกักข้อมูลส่วนบุคคลของคุณเท่าที่จำเป็นเพื่อบรรลุวัตถุประสงค์ในการรวบรวมข้อมูลดังกล่าว เว้นแต่จะมีกฎหมายกำหนดไว้เป็นอย่างอื่น หากคุณอาศัยอยู่ในประเทศเกาหลี โปรดพิจารณาข้อตกลงเฉพาะของประเทศต่างๆ
    • สิทธิเกี่ยวกับข้อมูลส่วนบุคคลของคุณ: คุณมีสิทธิที่จะขอ: (i) เข้าถึงข้อมูลส่วนบุคคลของคุณ; (ii) แก้ไขข้อมูลส่วนบุคคลของคุณให้ถูกต้องหากข้อมูลนั้นไม่ครบถ้วนหรือไม่ถูกต้อง; (iii) ลบข้อมูลส่วนบุคคลของคุณ นอกจากนี้ คุณยังมีสิทธิที่จะคัดค้านการประมวลผลข้อมูลส่วนบุคคลของคุณด้วยเหตุผลที่ถูกต้องตามกฎหมายในสถานการณ์พิเศษ เมื่อเราได้รับความยินยอมจากคุณในการประมวลผลข้อมูลส่วนบุคคลของคุณแล้ว คุณสิทธิในการถอนความยินยอมได้ทุกเมื่อ คุณยังมีสิทธิคัดค้านการประมวลผลข้อมูลส่วนบุคคลของคุณรวมถึงการเลือกไม่ให้ใช้ข้อมูลส่วนบุคคลเพื่อวัตถุประสงค์ตลาดแบบตรง หากคุณอาศัยอยู่ในประเทศอาร์เจนตินา หรือประเทศเกาหลี โปรดพิจารณาข้อตกลงเฉพาะของประเทศต่างๆ

Space Run รับและรวบรวมข้อมูลซึ่งอาจรวมถึงข้อมูลส่วนบุคคลจากบราวเซอร์ของคุณเมื่อคุณใช้เว็บไซต์ของเรา เราใช้วิธีการหลากหลายในการรวบรวมข้อมูลเหล่านี้ เช่น cookies และ pixel tags ซึ่งข้อมูลดังกล่าวอาจรวมถึง (i) IP Address ของคุณ; (ii) เครื่องมือระบุ cookie ที่มีลักษณะเฉพาะ, ข้อมูล cookie และข้อมูลว่าอุปกรณ์ของคุณมีซอฟต์แวร์ที่สามารถเข้าถึงคุณลักษณะเฉพาะบางอย่างได้หรือไม่; (iii) เครื่องมือระบุอุปกรณ์ที่มีลักษณะเฉพาะและประเภทของอุปกรณ์; (iv) โดเมน, ประเภทของบราวเซอร์ และภาษา; (v) ระบบปฏิบัติงานและการตั้งค่าระบบ; (vi) ประเทศและเขตเวลา; (vii) เว็บไซต์ที่เข้าชมก่อนหน้า; (viii) ข้อมูลเกี่ยวกับการใช้งานเว็บไซต์ของเรา เช่น การกดคลิก, การซื้อและความพึงพอใจที่ระบุไว้; และ (ix) เวลาที่เข้าใช้และ URLs ที่อ้างถึง
บุคคลที่สามอาจสามารถรวบรวมข้อมูลจากเว็บไซต์ผ่านทาง cookies, plug-ins ของบุคคลที่สาม และ widgets ได้เช่นกัน บุคคลที่สามเหล่านี้รวบรวมข้อมูลจากบราวเซอร์ของคุณโดยตรงและการประมวลผลของข้อมูลเหล่านี้จะเป็นไปตามนโยบายความเป็นส่วนตัวของพวกเขาเอง 
เราใช้ cookies และ pixel tags เพื่อติดตามความเคลื่อนไหวของลูกค้าของเราบนเว็บไซต์และเพื่อให้ทราบถึงความพึงพอใจของลูกค้าของเรา (อาทิ ประเทศและภาษาที่เลือก) สิ่งนี้ทำให้เราสามารถให้บริการและพัฒนาให้ลูกค้าได้รับประสบการณ์ออนไลน์ที่ดี เรายังใช้ cookies และ pixel tags เพื่อให้ได้มาซึ่งข้อมูลรวมเกี่ยวกับความแน่นหนาของผู้ใช้บนเว็บไซต์และการใช้เว็บไซต์ เพื่อทราบถึงกระแสและสถิติเพื่อใช้ในการพัฒนาเว็บไซต์ของเราต่อไป โดยทั่วไปแล้วมี cookies สามประเภทที่ใช้บนเว็บไซต์ของเรา 
    • การทำงาน: cookies ประเภทนี้มีความจำเป็นต่อระบบการทำงานพื้นฐานของเว็บไซต์ ดังนั้นจึงต้องเปิดใช้งานตลอดเวลา และยังรวมถึง cookies ที่ทำหน้าที่จดจำคุณในขณะที่คุณเยี่ยมชมเว็บไซต์ของเราภายในหนึ่งครั้งหรือหลายครั้งหากคุณขอ cookies เหล่านี้ช่วยให้ระบบตะกร้าสินค้าและเช็คเอาท์สามารถดำเนินไปได้และช่วยในเรื่องปัญหาด้ายความปลอดภัยและการปฏิบัติตามกฎระเบียบ
    • ผลงาน: cookies ประเภทนี้ทำให้เราสามารถพัฒนาระบบการทำงานของเว็บไซต์โดยวิธีการติดตามการใช้งาน ในบางกรณี cookies เหล่านี้ช่วยพัฒนาความเร็วในการประมวลคำขอของคุณและช่วยให้เราจดจำความพึงพอใจที่คุณเลือกไว้บนเว็บไซต์ การปฏิเสธ cookies เหล่านี้อาจส่งผลให้การให้คำแนะนำของเราไม่มีประสิทธิภาพและทำให้เว็บไซต์เราทำงานช้าลง
    • สังคมออนไลน์และโฆษณา: cookies ประเภทสังคมออนไลน์ทำให้สามารถเชื่อมต่อคุณไปยังเครือข่ายสังคมของคุณและการโพสต์เนื้อหาจากเว็บไซต์ของเราลงสังคมออนไลน์อื่น ๆ ส่วน cookies ประเภทโฆษณา (ของบุคคลที่สาม) ทำหน้าที่รวบรวมข้อมูลเพื่อให้การโฆษณาตรงตามความสนใจของคุณมากขึ้น ทั้งในและนอกเว็บไซต์ของเรา ในบางกรณี cookies เหล่านี้ครอบคลุมถึงการประมวลผลข้อมูลส่วนบุคคลของคุณด้วย หากปฏิเสธ cookies นี้อาจส่งผลให้คุณได้รับโฆษณาที่ไม่ตรงตามความสนใจของคุณหรือทำให้คุณไม่สามารถเชื่อมต่อกับ Facebook, Twitter หรือสังคมออนไลน์อื่น ๆ ได้อย่างมีประสิทธิภาพ และ/หรือไม่สามารถแชร์เนื้อหาลงในสังคมออนไลน์ได้
สำหรับข้อมูลที่ครอบคลุมและล่าสุดเกี่ยวกับการที่บุคคลที่สามจะสามารถเข้าถึงเว็บบราวเซอร์ของคุณในทุก ๆ ครั้ง (ผ่านทางเว็บไซต์ของ Space Run หรือทางอื่น) เราขอแนะนำให้คุณติดตั้ง plugin เว็บบราวเซอร์ที่สร้างขึ้นสำหรับวัตถุประสงค์นี้เฉพาะ คุณยังสามารถเลือกให้คอมพิวเตอร์ของคุณเตือนคุณในแต่ละครั้งที่ cookie ถูกส่งหรือเลือกปิดการใช้งาน cookies ทั้งหมด คุณสามารถทำทั้งหมดนี้ได้ผ่านทางหน้าการตั้งค่าบราวเซอร์ของคุณบนทุกบราวเซอร์และอุปกรณ์ที่คุณใช้ บราวเซอร์แต่ละอันจะมีความแตกต่างกันเล็กน้อย ดังนั้นให้คุณดูที่เมนูความช่วยเหลือเกี่ยวกับบราวเซอร์เพื่อเรียนรู้การแก้ไข cookies ของคุณ หากคุณปิดการใช้งาน cookies คุณอาจไม่สามารถเข้าถึงคุณลักษณะเฉพาะหลายอย่างที่ทำให้เว็บไซต์และแอพพลิเคชั่นของเรามีประสิทธิภาพ และบริการบางอย่างของเราก็จะไม่สามารถทำงานอย่างมีประสิทธิภาพเช่นกัน
การใช้เว็บไซต์และแอพพลิเคชั่นของ Space Run กับสินค้าและบริการของบุคคลที่สาม
เว็บไซต์และแอพพลิเคชั่นของเราทำให้คุณสามารถใช้งานสินค้าดิจิตอลและบริการที่หลากหลาย ตัวอย่างเช่น เว็บไซต์และแอพพลิเคชั่นของเราสามารถใช้งานร่วมกับอุปกรณ์ของบุคคลที่สามเพื่อการติดตามกิจกรรม, เครือข่ายสังคม, บริการเพลงออนไลน์และบริการดิจิตอลอื่นๆ
หากคุณเลือกที่จะเชื่อมต่อบัญชี Space Run ของคุณเข้ากับอุปกรณ์หรือบัญชีของบุคคลที่สาม สิทธิส่วนบุคคลของคุณบนแพลตฟอร์มของบุคคลที่สามจะต้องเป็นไปตามนโยบายของบุคคลที่สามเหล่านั้น ตัวอย่างเช่น หากคุณเลือกจะแชร์กิจกรรม Space Run กับสังคมออนไลน์แพลตฟอร์มของบุคคลที่สาม ข้อมูลที่อยู่บนแพลตฟอร์มจะอยู่ภายใต้บังคับของนโบายของแพลตฟอร์ม
เว็บไซต์และแอพพลิเคชั่นของเราอาจส่งมอบลิงค์ให้กับเว็บไซต์และแอพพลิเคชั่นบุคคลที่สามเพื่อความสะดวกหรือข้อมูลของคุณ เว็บไซต์และแอพพลิเคชั่นซึ่งถูกลิงค์จะมีการแจ้งเตือนหรือนโยบายความเป็นส่วนตัวของมันเอง ซึ่งเราสนับสนุนให้คุณตรวจสอบเสียก่อน เว็บไซต์หรือแอพพลิเคชั่นซึ่งถูกลิงค์ใดๆ ที่เราไม่ได้เป็นเจ้าของและไม่ได้ควบคุมจะไม่อยู่ในความรับผิดชอบของเราสำหรับเนื้อหา, การใช้เว็บไซต์หรือแอพพลิเคชั่น, หรือการบังคับใช้ความเป็นส่วนตัวของเว็บไซต์และแอพพลิเคชั่นนั้นๆ
การเปลี่ยนแปลงนโยบายความเป็นส่วนตัวของเรา
กฎหมายที่ใช้บังคับและแนวทางปฏิบัติของเราเปลี่ยนแปลงเป็นครั้งคราว หากเราตัดสินใจที่จะอัพเดทนโยบายของเรา เราจะโพสต์การเปลี่ยนแปลงดังกล่าวให้คุณทราบบนเว็บไซต์และแอพพลิเคชั่นของเรา หากเราทำการเปลี่ยนแปลงนโยบายเกี่ยวกับการประมวลผลข้อมูลส่วนบุคคลของคุณอย่างมีนัยสำคัญ เราจะแจ้งให้คุณทราบล่วงหน้า หรือหากกฎหมายบังคับ เราจะขอความยินยอมจากคุณก่อนที่จะดำเนินการเปลี่ยนแปลง เราสนับสนุนให้คุณอ่านนโยบายของเราและคอยติดตามแนวทางปฏิบัติของเรา นโยบายนี้ได้รับการแก้ไขครั้งล่าสุดเมื่อวันที่ มกราคม 2561 (2018)

หากคุณอาศัยอยู่ในประเทศใดประเทศหนึ่งด้านล่างนี้ ข้อตกลงเพิ่มเติมสำหรับประเทศนั้น ๆ จะถูกนำมาใช้และแทนข้อตกลงใดที่ขัดแย้งกับข้อตกลงในนโยบายความเป็นส่วนตัวนี้

ข้อย่อยเรื่อง “สิทธิเกี่ยวกับข้อมูลส่วนบุคคลของคุณ” ภายใต้หัวข้อย่อยเรื่อง “การคุ้มครองและจัดการข้อมูลส่วนบุคคลของคุณ” ได้รับการแก้ไขโดยการเพิ่มข้อความดังจะกล่าวต่อไปนี้ไว้ที่ส่วนท้ายของย่อหน้าแรก:
 “ผู้ครอบครองข้อมูลส่วนบุคคลมีสิทธิเข้าถึงข้อมูลนั้นอย่างอิสระภายในระยะเวลาไม่น้อยกว่าหกเดือน เว้นแต่จะสามารถพิสูจน์การมีส่วนได้เสียตามกฎหมายได้ตามมาตรา 14 ข้อ 3 พระราชบัญญัติ 25.326. หน่วยงานคุ้มครองข้อมูลแห่งชาติเป็นผู้มีอำนาจบังคับการตามพระราชบัญญัติ 25.326 และมีอำนาจดูแลรายงานและข้อเรียกร้องเกี่ยวกับการไม่ปฏิบัติตามเงื่อนไขการคุ้มครองข้อมูล เจ้าของข้อมูลส่วนบุคคลมีสิทธิเรียกให้เพิกถอนชื่อของผู้นั้นออกจากฐานข้อมูล”

ย่อหน้าต่อไปนี้ให้เพิ่มไว้ในข้อย่อยเรื่อง “การกักข้อมูลของคุณ” ภายใต้หัวข้อย่อยเรื่อง “การคุ้มครองและจัดการข้อมูลส่วนบุคคลของคุณ”:
“กล่าวโดยเฉพาะคือ ให้รวมถึง: (i) บันทึกเกี่ยวกับการเข้าถึงและการใช้เว็บไซต์และแอพพลิเคชั่น - สามเดือน; (ii) บันทึกเกี่ยวกับสัญญา, การทำธุรกรรม และการจัดหาสินค้า - ห้าปี; และ (iii) บันทึกเกี่ยวกับข้อร้องเรียนและข้อพิพาท - สามปี เราจะทำให้แน่ใจว่าบุคคลอื่นใดที่ประมวลผลข้อมูลส่วนบุคคลของคุณในนามของเราจะปฏิบัติตามระยะเวลาการกักข้อมูลเดียวกันนี้”
จุดนำย่อหน้าต่อไปนี้ให้เพิ่มไว้หลัง “การกักข้อมูลส่วนบุคคลของคุณ” ภายใต้หัวข้อย่อยเรื่อง “การคุ้มครองและจัดการข้อมูลส่วนบุคคลของคุณ”:
    • “การทำลายข้อมูลของคุณ: เราจะทำลายข้อมูลส่วนบุคคลของคุณโดยไม่ชักช้าเมื่อเราไม่มีความจำเป็นต้องกักข้อมูลนั้นไว้อีกต่อไป หากข้อมูลถูกพิมพ์ลงบนกระดาษ ข้อมูลนั้นจะถูกฉีกหรือเผา หากข้อมูลถูกเก็บไว้ในรูปแบบอิเล็กทรอนิกส์ ข้อมูลนั้นจะถูกทำลายอย่างถาวรโดยวิธีการทางเทคนิค”
ข้อย่อยเรื่อง “สิทธิเกี่ยวกับข้อมูลส่วนบุคคลของคุณ” ภายใต้หัวข้อย่อยเรื่อง “การคุ้มครองและจัดการข้อมูลส่วนบุคคลของคุณ” ให้ลบทิ้งและแทนทั้งหมดด้วยข้อความดังจะกล่าวต่อไปนี้: 
    • “สิทธิเกี่ยวกับข้อมูลส่วนบุคคลของคุณ: คุณมีสิทธิที่จะ (i) เข้าถึงข้อมูลส่วนบุคคลของคุณ; (ii) แก้ไขข้อมูลส่วนบุคคลของคุณให้ถูกต้องหากข้อมูลนั้นไม่ครบถ้วนหรือไม่ถูกต้อง; หรือ (iii) ลบข้อมูลส่วนบุคคลของคุณในเวลาใดก็ได้ คุณยังมีสิทธิที่จะขอให้พักการประมวลผลข้อมูลส่วนบุคคลของคุณตามกฎหมายที่เกี่ยวข้อง เมื่อเราได้รับความยินยอมจากคุณในการประมวลผลข้อมูลส่วนบุคคลของคุณแล้ว คุณมีสิทธิที่จะถอนความยินยอมได้ทุกเมื่อ คุณยังมีสิทธิจะคัดค้านการประมวลผลข้อมูลส่วนบุคคลของคุณรวมถึงการเลือกไม่ให้ใช้งานข้อมูลส่วนบุคคลเพื่อวัตถุประสงค์ตลาดแบบตรง”
ประเทศรัสเซีย
ข้อย่อยเรื่อง  “การโอนข้อมูลระหว่างประเทศ” ภายใต้หัวข้อย่อยเรื่อง “การคุ้มครองและจัดการข้อมูลส่วนบุคคลของคุณ” ให้ลบทิ้งและแทนทั้งหมดด้วยข้อความดังจะกล่าวต่อไปนี้:
    • “การโอนข้อมูลระหว่างประเทศ: ข้อมูลส่วนบุคคลของคุณที่เรารวบรวมหรือสร้างขึ้น (ประมวลผล) บนพื้นที่เว็บไซต์และแอพพลิเคชั่นของเราจะถูกเก็บรักษาไว้ในสหพันธรัฐรัสเซีย และสำเนาของข้อมูลนั้นสามารถโอนไปยังประเทศสหรัฐอเมริกาหรือประเทศอื่นๆ ได้”
